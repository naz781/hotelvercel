# Hotel Booking Web App

A **responsive and interactive hotel booking website** built with **React, Vite, Clerk, and React Router DOM**.  
Users can browse hotels, view details, book rooms, and track their past bookings through a dashboard.

---

## Features

- Browse hotels by city, rating, or price  
- View detailed hotel information and images  
- Book rooms securely via **Clerk Authentication**  
- Customer dashboard showing past bookings  
- Fully responsive design for desktop and mobile  

---

## Tech Stack

- **Frontend:** React.js, Vite  
- **Authentication:** Clerk  
- **Routing:** React Router DOM  
- **Styling:** Tailwind CSS

## Project Description

This project demonstrates a **full-featured hotel booking system** with:

- Secure user authentication via **Clerk**  
- Dynamic routing using **React Router DOM**  
- Interactive UI with responsive design  
- Dashboard for customers to view **past bookings**  

It is built for **portfolio purposes** and is **not publicly deployable** due to AI API credentials and private data handling.
